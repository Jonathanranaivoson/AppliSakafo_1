# AppliSakafo - Application de Gestion de Menu Alimentaires Domestique

##  Description
Application web Java/JSP pour la gestion des menus alimentaires dans un foyer domestique avec calculs et gestion automatiques des menus et recettes proposee.

##  Technologies
- **Backend**: Java avec POO (héritage, encapsulation, interfaces)
- **Frontend**: JSP pur (SANS JavaScript, SANS Servlet)
- **Base de données**: PostgreSQL
- **Serveur d'application**: Apache Tomcat
